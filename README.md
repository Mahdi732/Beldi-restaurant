# Beldi Restaurant Website

A modern restaurant website built with HTML and CSS.

## Overview

Beldi is a bakery and café website featuring a clean, elegant design with an intuitive user interface.

## Pages

- **Home** - Landing page with hero section
- **Menu** - Browse our delicious products (Cakes, Muffins, Croissants, Bread, Tarts)
- **Contact** - Get in touch with us
- **A Propos** - Learn about our story

## Technologies

- HTML5
- CSS3
- Font Awesome Icons
- Google Fonts (Sansita Swashed, Inter, Poppins)

## Features

- Responsive design
- Clean navigation
- Product gallery
- Contact form
- Social media integration

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your browser
